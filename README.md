- ft_isalpha
- ft_isdigit
- ft_isalnum
- ft_isascii
- ft_isprint
- ft_toupper
- ft_tolower

Functions to manipulate strings:
- ft_strlen
- ft_strlcpy
- ft_strlcat
- ft_strchr

Functions to manipulate memory:
- ft_memset

- ft_memcpy
- ft_memmove
- ft_memchr
